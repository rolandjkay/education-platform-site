$\pi$（「パイ」と読みます）という数を知らない人はほとんどいないでしょう。学校で数学を学んだなら、きっと覚えているはずです……3.142、でしたよね？　おそらく次のような「実用的な」問題を解くために、この数を暗記したのではないでしょうか：

> *半径10mの円形の畑の周の長さを求めなさい。*

$$\text{circumference} = 2 \times \pi \times 10 = 20 \times 3.142 = 62.84\text{ m}$$

卒業後の人生でたいへん役に立ったことでしょう、きっと。

でも、$\pi$とは*いったい何なのか*？　数学者が適当に作った数字にすぎないのでしょうか？

実は、真実はずっと面白いものです。誰もこの数を「発明」したわけでも、作り上げたわけでもありません。これは宇宙そのものの性質であり、驚くべきことに、原理的には紙とペン、コンパスと定規さえあれば、台所のテーブルで自分で計算——少なくとも推定——することができるのです。

さらに、$\pi$が3.142…でなければならない理由は*ない*のです。それはわたしたちの宇宙における値ですが、別の値になりえます。もしいつかあなたが多次元テレポーターをくぐり抜けることがあれば、どこかの台所のテーブルに座って$\pi$の値を推定してみてください。3.142…でなければ、別の宇宙に迷い込んでしまったとわかるはずです。

でもまず——$\pi$とは正確には何なのでしょうか？

## 正方形から始めよう

中心から一番近い辺までの距離が$r$である正方形の面積を求めたいとします。これは、一辺の長さが$2r$であると言うことと同じです。

<PiSquareDiagram />

正方形の面積は縦の長さ×横の長さですから、面積は：

$$A_{\text{square}} = 2r \times 2r = 4r^2$$

> **補足：** $r^2$とは$r \times r$のことです。たとえば$r = 3$なら、$r^2 = 3 \times 3 = 9$です。

## $\pi$は円に関係している

ここまで大丈夫ですか？　では次に、別の単純な形——円の面積を求めたい場合はどうでしょうか？　具体的には、半径$r$の円です：

<PiCircleDiagram />

円の面積も、*何らかの数*×$r^2$で表せそうだと想像できます。では、その数とは何でしょうか？

お気づきでしょうが、この謎の数を$\pi$と呼ぶことにして、それが何であるかを解き明かしてみましょう。

## $\pi$の値を推定する

学校では次の公式を習ったはずです：

$$A_{\text{circle}} = \pi \times r^2$$

そして$\pi$の値を暗記しました。でも、その値は誰かに*教えてもらった*ものです。自分で導き出したいとしたら、どうすればよいでしょうか？

### ステップ1：正方形の中の円

半径$r$の円と、一辺$2r$の正方形を、円が正方形にぴったり収まるように重ねて描いてみましょう：

<PiCircleInSquareDiagram />

円は正方形よりも小さな面積を占めています。正方形の面積は$4r^2$で、円はその中に収まっていますから、円の面積は$4r^2$*より小さい*はずです。つまり：

$$\pi < 4$$

これで$\pi$は4より小さいとわかりました。$\pi$の「上限」を設けたわけです。では、下限は見つけられるでしょうか？

### ステップ2：円の中の正方形

今度は別の正方形を描いてみましょう——今度は円の*内側*に、頂点が円に接するように：

<PiSquareInCircleDiagram />

この内側の正方形の一辺の長さはいくつでしょうか？　（三平方の定理を使うと）一辺の長さは$r\sqrt{2}$となり、面積は$2r^2$になります。

<details>
<summary>三平方の定理による計算を見る</summary>

内側の正方形の各頂点は円の上にあり、中心から$r$の距離にあります。内側の正方形は外側の正方形に対して45°回転していますから、その4分の1を見ると、二辺が等しく斜辺が$r$の直角三角形になります。

三平方の定理（$a^2 + b^2 = c^2$）により、二つの短い辺をどちらも$s$とすると：

$$s^2 + s^2 = r^2 \quad\Rightarrow\quad 2s^2 = r^2 \quad\Rightarrow\quad s = \frac{r}{\sqrt{2}}$$

内側の正方形の一辺の全長は$2s = \frac{2r}{\sqrt{2}} = r\sqrt{2}$ですから、その面積は：

$$A_{\text{inner square}} = \left(r\sqrt{2}\right)^2 = 2r^2$$

</details>

円はこの内側の正方形を完全に含んでいますから、円の面積は$2r^2$*より大きい*はずです：

$$2 < \pi$$

二つの限界をまとめると：

$$2 < \pi < 4$$

絞り込めてきました！　でも、まだ範囲がかなり広いですね。もっとうまくできるでしょうか？

### ステップ3：辺の数を増やすとより正確になる

ここが重要な気づきです：わたしたちは正方形（4辺の形）を使って大まかな推定をしました。*もっと多くの*辺を持つ形を使ったらどうでしょうか？

内外の正方形の代わりに、内外の**六角形**（6辺）を使うことができます。内側の六角形は頂点が円に接するよう円の内側に収まり、外側の六角形は各辺の中点が円に接するよう円の外側に置かれます——正方形でやったのと同じやり方です。

六角形は正方形よりも円をたくさん埋めますから、内外の限界の差が小さくなります。

<PiHexagonDiagram />

面積を計算すると（少しの三角法を使って）：

$$3.000 < \pi < 3.464$$

上々です！　では**八角形**（8辺）を試してみましょう：

$$2.828 < \pi < 3.314$$

**十二角形**（12辺）では：

$$3.000 < \pi < 3.215$$

パターンが見えてきましたか？　辺の数を増やすにつれて、内側の多角形はどんどん大きくなり（内側から円に近づき）、外側の多角形はどんどん小さくなります（外側から近づく）。二つの限界は互いに近づき、$\pi$をどんどん狭い範囲に閉じ込めていきます。

| 多角形の辺の数 | 下限 | 上限 | 差 |
|---|---|---|---|
| 4（正方形） | 2.000 | 4.000 | 2.000 |
| 6（六角形） | 3.000 | 3.464 | 0.464 |
| 8（八角形） | 2.828 | 3.314 | 0.486 |
| 12 | 3.000 | 3.215 | 0.215 |
| 24 | 3.106 | 3.160 | 0.054 |
| 48 | 3.133 | 3.146 | 0.013 |
| 96 | 3.139 | 3.143 | 0.003 |

96辺になると、$\pi$を3.139から3.143の間に絞り込めます。そして無限大まで進めば——無限に多くの辺を持つ多角形は*まさに*円です——正確に$\pi = 3.14159265...$という値が得られます。

これは本質的に、紀元前250年頃にアルキメデスが行ったことです。彼は96辺の多角形を使い、次のことを導き出しました：

$$3\frac{10}{71} < \pi < 3\frac{1}{7}$$

つまり$3.1408... < \pi < 3.1429...$です。

電卓が存在する2000年以上前に、幾何学と忍耐だけを使って、これほどの結果を出したのですから、たいしたものです。

## $\pi$はどこにでも現れる

$\pi$が何かわかると、いたるところで見かけるようになります。円の面積だけの話ではありません。

この記事の冒頭で触れた、円の**周の長さ**は：

$$C = 2\pi r$$

半径$r$の**球の体積**は：

$$V = \frac{4}{3}\pi r^3$$

統計学の中心にある釣り鐘型の曲線、**正規分布**には、$\pi$がその公式に組み込まれています：

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \, e^{-\frac{(x - \mu)^2}{2\sigma^2}}$$

$\pi$は振り子の揺れ、ギターの弦の振動、惑星の軌道、量子力学の方程式にも現れます。円形のもの、周期的なもの、波のようなもの、どこにでも$\pi$は潜んでいます。

## $\pi$は終わらない

$e$と同様に、$\pi$は**無理数**です——その小数展開は繰り返しなく永遠に続きます：

$$\pi = 3.14159265358979323846...$$

単純な分数で表すことは絶対にできません。桁はただ続いていきます。2024年時点で、$\pi$は100兆桁以上計算されています——そんなに必要な人は誰もいないのに（観測可能な宇宙の周長を水素原子の幅以内の精度で計算するには、約40桁あれば十分です）、それでも計算されるのは、コンピューターの性能を測るよいベンチマークになるからです。そして正直に言えば、数学者というのはちょっと執念深いものだからです。

## $\pi$は別の値になりえるか？

最初に、別の宇宙では$\pi$の値が違うかもしれないと言いましたね？　その意味を説明しましょう。

この記事でやってきたことはすべて、*平らな*面の上で作業していることを前提としています。でも、もし空間が平らでなかったら？

球の表面——たとえば地球の表面——に円を描くことを想像してください。コンパスの針を北極に置いて、その周りに円を描きます。この円の内側にある地球の表面積は、この円を通るように刃を入れて地球を切断したときに得られる円盤の面積よりも大きくなります。

下のアニメーションでこれを確認できます。藍色の面は球の表面に描いた円の面積を示しています。赤い円盤は同じ周長を持つ平らな円の面積を示しています。藍色の面の面積は赤い円盤の面積より大きく、そのため$\pi$は3.142より大きくなります。
<SphericalCapExplorer />

3.14159...という値は**平らな**空間に特有のものです。曲がった空間ではその値が変わります。これは単なる抽象的な数学ではありません——アインシュタインの一般相対性理論は、質量を持つ物体が周囲の空間を実際に曲げることを教えてくれます。ブラックホールの近くで、その歪んだ空間の中で円を測定できたとしたら、$\pi$の異なる値が測定されるでしょう。

ですから、もしいつかテレポーターをくぐり抜けることがあれば——何をすべきかもうわかりますね。

## まとめ

$\pi$は最初、単純に見えます——ただの3.142、でしょう？　でも実際は、幾何学と確率、円と波、単純な測定と宇宙の最も深い構造を結びつける、平らな空間の基本定数なのです。そして2000年以上前にアルキメデスが示してくれたように、定規とコンパスと十分な忍耐さえあれば、自分でそれを追い求めることができます。
