大多数人都听说过数字 $\pi$（读作"派"）。如果你在学校学过数学，你可能还记得……3.142，对吧？你很可能背下了这个数字，以便解决这类实用问题：

> *计算半径为10米的圆形田地的周长。*

$$\text{周长} = 2 \times \pi \times 10 = 20 \times 3.142 = 62.84\text{ 米}$$

这在你毕业之后的生活里一定大有用处，对吧？

但是，$\pi$ 到底*是什么*？只是数学家随便编出来的某个数字吗？

其实，真相要有趣得多。这个数字并非任何人"发明"或凭空捏造的。它是宇宙本身的一个属性，而且令人惊奇的是，你原则上可以自己计算它——或者至少估算它——只需坐在厨房桌旁，拿着纸笔、圆规和直尺就够了。

更妙的是，它*不一定*等于3.142……那是它在我们这个宇宙中的值，但在别处可能截然不同。假如有一天你偶然穿越了一台多维传送器，不妨坐到某张厨房桌旁，估算一下 $\pi$ 的值。如果它不等于3.142……那你就知道自己误入了一个平行宇宙。

但首先——$\pi$ 究竟是什么？

## 从正方形说起

想象一下，我们想计算一个正方形的面积，其中心到最近一条边的距离为 $r$。这等同于说，一条边的长度为 $2r$。

<PiSquareDiagram locale={locale} />

我们知道正方形的面积等于一条边长乘以另一条边长，所以面积为：

$$A_{\text{正方形}} = 2r \times 2r = 4r^2$$

> **小提示：** $r^2$ 就是 $r \times r$。所以如果 $r = 3$，那么 $r^2 = 3 \times 3 = 9$。

## $\pi$ 与圆密不可分

目前为止还清楚吧？现在，如果我们想计算另一种简单形状——圆的面积呢？具体来说，是半径为 $r$ 的圆：

<PiCircleDiagram locale={locale} />

我们可以猜想，圆的面积也等于*某个数字*乘以 $r^2$。但那个数字是什么呢？

你大概已经猜到了，我们要把这个神秘数字叫做 $\pi$，然后设法弄清楚它的值。

## 估算 $\pi$ 的值

在学校里，你学过这个公式：

$$A_{\text{圆}} = \pi \times r^2$$

并且背下了 $\pi$ 的值。但那个值是别人*告诉*你的。如果你想自己推导出来，该怎么做呢？

### 第一步：正方形内的圆

让我们把半径为 $r$ 的圆和边长为 $2r$ 的正方形画在一起，让圆刚好内切于正方形：

<PiCircleInSquareDiagram locale={locale} />

圆占用的空间比正方形小。正方形的面积是 $4r^2$，而圆在其内部，所以圆的面积必然*小于* $4r^2$。这意味着：

$$\pi < 4$$

由此我们知道 $\pi$ 必然小于4；我们为 $\pi$ 的值设定了一个"上界"。我们能找到下界吗？

### 第二步：圆内的正方形

现在再画一个正方形——这次*在圆的内部*，让它的顶角恰好落在圆上：

<PiSquareInCircleDiagram locale={locale} />

这个内接正方形的边长是多少？利用勾股定理可以得出，边长为 $r\sqrt{2}$，因此面积为 $2r^2$。

<details>
<summary>展示勾股定理的推导过程</summary>

内接正方形的每个顶角都位于圆上，距圆心距离为 $r$。内接正方形相对于外接正方形旋转了45°，所以如果我们看其中四分之一，会得到一个等腰直角三角形，两条直角边相等，斜边长为 $r$。

根据勾股定理（$a^2 + b^2 = c^2$），若两条短边都为 $s$，则：

$$s^2 + s^2 = r^2 \quad\Rightarrow\quad 2s^2 = r^2 \quad\Rightarrow\quad s = \frac{r}{\sqrt{2}}$$

内接正方形的完整边长为 $2s = \frac{2r}{\sqrt{2}} = r\sqrt{2}$，所以其面积为：

$$A_{\text{内接正方形}} = \left(r\sqrt{2}\right)^2 = 2r^2$$

</details>

圆完全包含这个内接正方形，所以圆的面积必然*大于* $2r^2$：

$$2 < \pi$$

将两个界合在一起：

$$2 < \pi < 4$$

我们在逐步缩小范围！但区间仍然相当宽。我们能做得更好吗？

### 第三步：边数越多，估算越精确

这里有一个关键的洞见：我们使用了正方形（四边形），得到了一个粗略的估算。如果我们使用边数*更多*的形状呢？

不用内外两个正方形，我们可以改用内外两个**六边形**（6条边）。内接六边形的顶角落在圆上，外切六边形的每条边中点与圆相切——正如我们对正方形所做的那样。

六边形比正方形能填充更多的圆的面积，因此内外两个界之间的差距会缩小。

<PiHexagonDiagram locale={locale} />

如果我们计算面积（需要用到一些三角函数），得到：

$$3.000 < \pi < 3.464$$

更好了！现在来试试**八边形**（8条边）：

$$2.828 < \pi < 3.314$$

再来试**十二边形**（12条边）：

$$3.000 < \pi < 3.215$$

看出规律了吗？随着边数的增加，内接多边形越来越大（从内部逼近圆），外切多边形越来越小（从外部逼近圆）。两个界不断收紧，将 $\pi$ 夹在越来越窄的范围内。

| 多边形边数 | 下界 | 上界 | 差值 |
|---|---|---|---|
| 4（正方形） | 2.000 | 4.000 | 2.000 |
| 6（六边形） | 3.000 | 3.464 | 0.464 |
| 8（八边形） | 2.828 | 3.314 | 0.486 |
| 12 | 3.000 | 3.215 | 0.215 |
| 24 | 3.106 | 3.160 | 0.054 |
| 48 | 3.133 | 3.146 | 0.013 |
| 96 | 3.139 | 3.143 | 0.003 |

当我们用到96条边时，已经把 $\pi$ 锁定在3.139到3.143之间。如果能让边数趋向无穷——一个有无穷多条边的多边形就是一个圆——我们就会精确得到 $\pi = 3.14159265...$

这基本上就是阿基米德在公元前250年左右所做的事。他使用了96边形，推导出：

$$3\frac{10}{71} < \pi < 3\frac{1}{7}$$

即 $3.1408... < \pi < 3.1429...$

对于一个仅凭几何和耐心，在计算器问世两千多年前就进行这项工作的人来说，这相当了不起。

## $\pi$ 无处不在

一旦你理解了 $\pi$ 是什么，你就会开始在各处发现它的身影。它不仅仅关乎圆的面积。

圆的**周长**（也是本文的起点）为：$$C = 2\pi r$$

半径为 $r$ 的**球体积**为：$$V = \frac{4}{3}\pi r^3$$

**正态分布**——统计学核心的钟形曲线——在其公式中内嵌了 $\pi$：

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \, e^{-\frac{(x - \mu)^2}{2\sigma^2}}$$

$\pi$ 出现在摆的摆动、吉他弦的振动、行星的轨道以及量子力学方程中。凡是存在圆形、周期性或波动性的地方，$\pi$ 都潜伏在附近。

## $\pi$ 永无止尽

和 $e$ 一样，$\pi$ 是**无理数**——它的小数展开永无止境，没有循环节：

$$\pi = 3.14159265358979323846...$$

你永远无法将其写成一个简单的分数。数字永远延续下去。截至2024年，$\pi$ 已被计算到超过100万亿位小数——并不是因为有人真的需要那么多位（计算可观测宇宙的周长精确到氢原子直径以内，只需大约40位），而是因为它是检验计算能力的一个好基准。而且，坦率地说，因为数学家都有一点执着劲。

## $\pi$ 可能不同吗？

还记得文章开头我们说 $\pi$ 在另一个宇宙中可能不同吗？我们的意思如下。

我们在本文中所做的一切，都假设是在*平坦的*平面上运算。但如果空间不是平坦的呢？

想象在球面上画一个圆——比如地球表面。把圆规尖放在北极，画一个围绕它的圆。地球表面被这个圆围起来的那部分，比我们用一把刀沿这个圆切过地球所得到的圆形截面要大。

你可以在下面的动画中看到这一点。靛蓝色的曲面显示了在球面上画出的圆的面积。红色圆盘显示了具有相同周长的平面圆的面积。靛蓝色曲面的面积大于红色圆盘的面积，因此在这里 $\pi$ 大于3.142。

<SphericalCapExplorer locale={locale} />

3.14159...这个值只适用于**平坦**空间。在弯曲空间中，它会发生变化。这不仅仅是抽象数学——爱因斯坦的广义相对论告诉我们，大质量天体确实会使周围的空间弯曲。在黑洞附近，如果你能在那扭曲的空间中测量圆，你会测出一个不同的 $\pi$ 值。

所以，如果你真的有一天踏入了那台传送器——你知道该怎么做了。

## 轮到你了

$\pi$ 乍看简单——不就是3.142嘛？但它实际上是平坦空间的一个基本常数，将几何与概率相连，将圆与波相连，将简单的测量与宇宙最深层的结构相连。正如阿基米德在两千多年前向我们展示的那样，你可以凭借一把直尺、一个圆规和无尽的耐心，亲手去追寻它。
